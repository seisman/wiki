vscode
======

homepage: https://code.visualstudio.com/

Extensions
----------

Search these extensions by name in https://marketplace.visualstudio.com.

Support More Languages
~~~~~~~~~~~~~~~~~~~~~~~

- **Better TOML**: Syntax highlight for TOML language
- **Python**: Python Support
- **reStructuredText**: reStructuredText support, also support live previewing sphinx documentation
- **C/C++**: C/C++ Support
- **C++ Intellisense**:
- **CMake**: CMake support

Better Display
~~~~~~~~~~~~~~

- **filesize**: Show the current file size in the status bar
- **TODO Highlight**: highlight TODOs, FIXMEs, and any keywords, annotations
- **Bracket Pair Colorizer**: Colorizing matching brackets
- **VSCode Great Icons**: Icons for different filetypes

Improve Efficiency
~~~~~~~~~~~~~~~~~~

- **Path Intellisense**: Autocompletes filenames.
- **GitHub Pull Requests**: Manage GitHub pull requests in vscode

Management
~~~~~~~~~~

- **Settings Sync**: Synchronize Settings, Snippets, Themes, File Icons, Launch, Keybindings,
  Workspaces and Extensions Across Multiple Machines Using GitHub Gist.

Others
~~~~~~

- **vscode-pdf**: Display PDF in VSCode
