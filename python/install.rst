Install Python packages
=======================

Install
-------

Using conda provided by Anaconda::

    conda install numpy

Using pip::

    pip install numpy

Install source code from GitHub::

    pip install git+https://github.com/seisman/HinetPy.git

Compile and install::

    python setup.py install

Upgrade
-------

Using conda::

    conda update --all

Using pip::

    pip install -U packagename
